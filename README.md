
# VintedLister

VintedLister is an AI-powered tool that helps you quickly create attractive listings for your used clothes on Vinted.com. Simply upload photos of your clothing items, and our AI will generate detailed descriptions including titles, categories, conditions, and suggested prices.

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/mhurhangee/vinted-lister.git
   ```

2. Install dependencies:

```shellscript
cd vinted-lister
npm install
```


3. Create a `.env.local` file with your OpenAI API key:

```plaintext
OPENAI_API_KEY=your_api_key_here
```


4. Run the development server:

```shellscript
npm run dev
```


5. Open [http://localhost:3000](http://localhost:3000) in your browser.


## Usage

1. Navigate to the Listing Generator
2. Upload up to 6 images of your clothing item
3. Tag each image appropriately (front, back, label, etc.)
4. Click "Create Listing from Images"
5. Copy the generated content to use in your Vinted listing
